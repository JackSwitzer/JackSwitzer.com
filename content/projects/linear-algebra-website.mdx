# Linear Algebra Learning Platform

An interactive educational website built while TAing APSC 174 Linear Algebra at Queen's University.

## Purpose

Created this platform to supplement in-class tutorials and provide students with 24/7 access to course materials, detailed explanations, and practice problems.

## Features

### Interactive Content
- **Tutorial Notes**: Detailed explanations for each week's topics
- **Embedded Videos**: Visual explanations of key concepts
- **Practice Problems**: Weekly problem sets with solutions

## Impact

Used as primary resource for 80+ students across 2 tutorial sections, providing consistent, searchable reference content alongside in-person instruction.
