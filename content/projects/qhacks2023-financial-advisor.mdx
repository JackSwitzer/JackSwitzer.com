# Virtual Financial Advisor

An all-in-one virtual financial advisor built at QHacks 2024, combining sentiment analysis, portfolio management, and AI-powered insights.

## Core Features

### Sentiment Analysis Engine
Built a web scraping system that:
- Scrapes thousands of financial news sites and social media
- Analyzes text for positive/negative sentiment
- Generates company-specific sentiment scores
- Tracks sentiment changes over time

### Portfolio Management
Integrated portfolio tracking capabilities:
- Monitor stock holdings and performance
- Receive AI-powered insights on positions
- Get alerts based on sentiment shifts

### ChatGPT Integration
Leveraged OpenAI's ChatGPT API to:
- Answer natural language financial questions
- Explain market concepts
- Provide conversational portfolio analysis

## Technical Stack

### Backend
- **Flask**: Python web framework for API endpoints
- **Web Scraping**: BeautifulSoup/requests for data collection
- **NLP**: Sentiment analysis processing pipeline

### AI Integration
- **ChatGPT API**: Natural language interface
- **Custom Prompts**: Financial domain-specific prompt engineering

## Hackathon Context

Built during QHacks at Queen's University, demonstrating rapid prototyping of an AI-powered financial tool.
