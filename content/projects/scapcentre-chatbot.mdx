# Scapcentre AI Chatbot

A consulting project building an AI-powered chatbot for Scapcentre's healthcare website.

## Project Overview

Scapcentre provides healthcare treatment services. This project aims to improve the customer experience by implementing an intelligent chatbot that can:

- Answer common questions about available treatments
- Guide users through the booking process
- Provide information for both new and returning customers
- Reduce friction in the customer journey

## Technical Approach

Building a conversational AI interface that integrates with the existing scapcentre.ca website to provide real-time assistance to visitors.

## Status

Currently in development - will be publicly visible once deployed to the main website.
